## üöÄ Fitur Utama
- **Real-time Price:** Pantau harga TON dalam USD & IDR langsung dari CoinGecko API.
- **Auto-Generating Chart:** Mengirim grafik pergerakan harga secara otomatis menggunakan `matplotlib`.
- **Smart Alert:** Bot hanya akan mengirimkan notifikasi jika terjadi perubahan harga (Naik/Turun).
- **Signal Indicator:** Memberikan sinyal sederhana (Buy/Sell/Hold) berdasarkan fluktuasi harga.
- **Low Latency:** Dibangun dengan sistem threading agar monitoring berjalan di background tanpa mengganggu command lain.
